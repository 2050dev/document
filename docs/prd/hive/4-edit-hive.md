# 编辑蜂巢信息、编辑意向信息

## 功能说明

本功能从前台单独蜂巢页面/单独意向页面的“编辑蜂巢信息/编辑意向信息”按钮进入，或者从后台-蜂巢管理/意向管理页面当中的“编辑”按钮进入。
