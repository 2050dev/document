# 单独蜂巢页面、单独意向页面

## 页面说明

在蜂巢首页点击一个蜂巢的logo图或者标题，可以进入一个单独蜂巢页。

在意向首页点击一个意向的logo图或者标题，可以进入一个单独意向页。

## 一个蜂巢涉及的数据

- 基础信息
  - 蜂巢名称
  - 蜂巢介绍
  - 蜂巢logo
  - 蜂巢owner
- 关联信息
  - 蜂巢召集人
  - parent蜂巢、children蜂巢
  - 蜂巢相关的动态
  - 蜂巢发起的活动
