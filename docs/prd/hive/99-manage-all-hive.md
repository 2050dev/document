# 后台-蜂巢管理、意向管理

## 功能说明

蜂巢管理后台包含页面：

1. 所有蜂巢
2. 新建蜂巢
3. 编辑蜂巢列表排序
4. 编辑蜂巢首页自定义信息
5. （可选）蜂巢动态历史记录
